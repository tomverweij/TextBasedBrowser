Description

Every browser accepts a string from the user and then displays a web page. The string from the user is a URL (Uniform Resource Locator) and looks something like this: https://www.google.com. After receiving the URL, the browser has a lot of work to do, but in a nutshell, this work results in finding the web page. It is located somewhere on the Internet, and the browser retrieves it. Since the https://www. part is always the same, it is often omitted and the link is shortened to google.com.

In the first stage, we'll try to imitate this behavior. In the source code, you will find two predefined variables that contain news texts. You will need to print their content as a response to the corresponding input URL.

Objectives

In the first stage, your program should:

  1. Accept a URL from the user as input.
    1. If the user input bloomberg.com, print the content of the bloomberg_com variable.
    2. If the user input nytimes.com, print the content of the nytimes_com variable.
  2. Keep on prompting the user for input until they type exit.
Example

The greater-than symbol followed by a space (> ) represents the user input. Note that it's not part of the input.

> bloomberg.com

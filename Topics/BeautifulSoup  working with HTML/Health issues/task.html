Health issues

For example, for this input and letter L (here we provide the link to the archive for consistency's sake):

http://web.archive.org/web/20201201053628/https://www.who.int/health-topics

the output should be:

["Landslides", "Lassa fever", "Leishmaniasis", "Leprosy", "Lymphatic filariasis"]

Tip: There is no need to overcomplicate the solution by difficult operations with tags, the problem can be solved by just going through the text paragraphs and picking the ones between Schistosomiasis and Sustainable Development Goals since these are the first and the last topics that should be in your list.
